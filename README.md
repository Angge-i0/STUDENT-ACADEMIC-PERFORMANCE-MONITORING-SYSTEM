# STUDENT-ACADEMIC-PERFORMANCE-MONITORING-SYSTEM
This study presents a Student Academic Performance Monitoring System designed to analyze, classify, and predict student outcomes using synthetically generated data and machine-learning techniques. The system aims to address the limitations of manual and reactive academic monitoring by developing a data-driven pipeline capable of identifying performance patterns and academic risk early. Using Python in Google Colab, a dataset of 15,000 synthetic Computer Engineering student profiles was created and preprocessed into wide, long, and modeling-ready formats. Three machine learning models were implemented: Random Forest and Gradient Boosting for supervised classification of performance categories, and K-Means for unsupervised clustering of behavioral and academic patterns. Results show that the Random Forest classifier achieved the highest accuracy at 85.33%, followed by Gradient Boosting at 83.49%. Feature importance analysis revealed that yearly GWAs, study hours, and attendance rate were the strongest predictors of performance. K-Means clustering produced four meaningful student groups, supported by a silhouette score of 0.36, indicating moderately distinct academic behavior patterns. These analytical outputs were integrated into an interactive Power BI dashboard, enabling clear visualization of trends, risk indicators, and model predictions. Overall, the system demonstrates that combining synthetic data, machine learning models, and interactive visual analytics can provide reliable insights for early intervention, personalized academic support, and data-informed decision-making in educational institutions.
